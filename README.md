# HashMap-OdinProject

A simple implementation of a hash map (hash table) in JavaScript using buckets and a basic hash function. Supports basic operations like set, get, remove, and automatically resizes when the load factor exceeds a threshold.

Supports the following methods:

• set(key, value) - Adds or updates a key-value pair.

• get(key) - Retrieves the value for a given key, or null if not found.

• has(key) - Checks if a key exists in the hash map.

• remove(key) - Removes a key-value pair.

• length() - Returns the number of stored key-value pairs.

• clear() - Clears all entries.

• keys() - Returns an array of all keys.

• values() - Returns an array of all values.

• entries() - Returns an array of [key, value] pairs.

• currentLoadFactor() - Returns the current load factor of the hash map.
